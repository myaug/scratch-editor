Block Level Filtering Test

Test Instructions

To test the block filtering functionality:

  1. Open the Scratch editor at http://localhost:8601
  2. Look for the Level menu in the top menu bar (Khám phá / Sáng tạo / Thành thạo / Studio)
  3. Select different levels and observe the blocks in each category
  4. Check that only the appropriate blocks show for each level

Expected Block Filtering Behavior

Explorer Level (Khám phá)

Motion: motion_movesteps, motion_turnright, motion_turnleft
Looks: looks_say, looks_show, looks_hide
Events: event_whenflagclicked, event_whenkeypressed

Creator Level (Sáng tạo)

Motion: All explorer blocks + motion_goto, motion_gotoxy
Looks: All explorer blocks + looks_switchcostumeto, looks_nextcostume
Sound: sound_play, sound_playuntildone

Master Level (Thành thạo)

Motion: All creator blocks + motion_pointindirection, motion_changexby
Control: control_wait, control_repeat, control_forever, control_if
Sensing: sensing_touchingobject, sensing_keypressed

Studio Level

Shows all blocks (no filtering applied)
